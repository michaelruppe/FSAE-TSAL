# FSAE-TSAL
A comparator-based TSAL design for FSAE Electric 2019

## Specifications

| Specificaions              |                   |
|----------------------------|-------------------|
| Max Voltage (HVin)         | 200V DC           |
| Switching threshold (HVin) | 60V DC            |
| Supply Voltage             | 12V DC            |
| Output                     | Switched Contacts |
| HVin < 60V                 | OPEN              |
| HVin > 60V                 | CLOSED            |
